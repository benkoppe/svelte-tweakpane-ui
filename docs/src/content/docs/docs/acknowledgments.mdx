Svelte Tweakpane UI was created by [Eric Mika](https://ericmika.com). The library really just stitches together a number of other open-source projects, credited below.

## Tweakpane

The [Tweakpane](https://tweakpane.github.io/docs/) library was created by [Hiroki Kokubun](https://cocopon.me).

## Tweakpane Plugins

Svelte Tweakpane UI incorporates the following Tweakpane plugins:

- [CameraKit](https://github.com/tweakpane/plugin-camerakit) by [Hiroki Kokubun](https://cocopon.me)
- [Essentials](https://github.com/tweakpane/plugin-essentials) by [Hiroki Kokubun](https://cocopon.me)
- [Image](https://github.com/metehus/tweakpane-image-plugin) by [Florian Morel](http://ayamflow.fr), [Matheus Dias](https://www.linkedin.com/in/matheusdbs/), and [Palash Bansal](https://github.com/repalash)
- [Profiler](https://github.com/0b5vr/tweakpane-plugin-profiler) by [0b5vr](https://0b5vr.com)
- [Rotation](https://github.com/0b5vr/tweakpane-plugin-rotation) by [0b5vr](https://0b5vr.com)
- [Textarea](https://github.com/panGenerator/tweakpane-textarea-plugin) by [Krzysztof Goliński](http://www.golinski.org) and [Jakub Koźniewski](https://pangenerator.com)
- [Waveform](https://github.com/shoedler/tweakpane-plugin-waveform) by [Simon Schödler](https://shoedler.github.io)

## Svelte

The [Svelte](https://svelte.dev) project was initated by [Rich Harris](https://twitter.com/rich_harris) and is developed and maintained by a [community of contributors](https://github.com/sveltejs/svelte/graphs/contributors).
